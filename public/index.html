Live DDoS Map

This application showcases the power of ScyllaDB in handling and analyzing high-intensity cyber activities. It's specifically designed to simulate tracking of DDoS (Distributed Denial of Service) attacks, demonstrating ScyllaDB's capability to efficiently manage thousands of operations every second. The core feature involves recording a large volume of simulated DDoS events into ScyllaDB, which can then be analyzed and visualized through real-time data retrieval from the same database.

Take a look at the various ScyllaDB performance metrics, noting the high performance and predictable low latency achievable with just one node. The performance you get from one node will scale linearly as you add more nodes with the same predictable results.
